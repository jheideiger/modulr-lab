→ le fond

le modulaire n'est pas un instrument. c'est un langage.

On dit souvent que la synthèse modulaire est complexe. Ce n'est pas faux. Mais ce n'est pas la bonne entrée. La vraie question n'est pas « est-ce que c'est dur ? » — c'est « est-ce que ça vaut le coup de désapprendre ce qu'on sait sur la musique pour redécouvrir le son autrement ? »

Un synthétiseur classique vous propose un chemin. Un oscillateur, un filtre, un amplificateur, une sortie. Vous tournez des boutons, vous obtenez des sons. Le modulaire, lui, vous tend une pile de câbles et vous dit : à vous de décider ce qui va où.

Ce n'est pas plus difficile. C'est différent. C'est une manière de penser le son où chaque connexion est un choix, chaque patch est une architecture, chaque session est une conversation entre vous et la machine.

→ la forme

pas de jargon gratuit. pas de gate-keeping.

La communauté modulaire est passionnée, mais elle peut être intimidante. Entre les acronymes (VCO, VCA, LFO, EG, S&H…), les débats d'initiés et les setups à cinq chiffres, le débutant curieux se retrouve vite à la porte.

Modulr lab prend le contre-pied. Ici, on explique les bases sans présupposer que vous savez déjà. On nomme les choses simplement. On préfère un exemple clair à une définition savante. Et on assume que la meilleure porte d'entrée, c'est la curiosité — pas le budget.

Ce site est gratuit, ouvert, sans publicité. Il est né d'une envie simple : partager ce qui m'a passionné, avec celles et ceux qui hésitent encore à franchir le pas.

→ la suite

quatre leçons pour commencer. le reste viendra.

Le cours que vous trouverez ici est un point de départ : quatre leçons vidéo accompagnées de textes, pour comprendre ce qu'est un signal, comment naît un son, à quoi servent les modules fondamentaux, et comment tout ça s'articule dans un premier patch.

À la fin, vous ne serez pas un expert. Mais vous aurez les clés pour décider si cette pratique est faite pour vous — et si oui, par où continuer.
